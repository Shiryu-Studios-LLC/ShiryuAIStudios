# Contributing to Shiryu AI Studio

Thank you for your interest in contributing to Shiryu AI Studio!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone --depth 1 https://github.com/YOUR_USERNAME/ShiryuAIStudios.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Test your changes: `npm run compile`
7. Commit and push

## Development Workflow

```bash
# Install dependencies
npm install

# Compile the project
npm run compile

# Run in development mode
npm run watch

# Run tests
npm run test-node
```

## Coding Guidelines

- Follow the existing code style (TypeScript, ESLint configured)
- Keep changes focused and minimal
- Write clear commit messages
- Test your changes before submitting

## Pull Request Process

1. Update documentation if needed
2. Ensure the build compiles without errors
3. Write a clear description of your changes
4. Reference any related issues

## Reporting Issues

- Use the GitHub Issues tracker
- Include steps to reproduce the issue
- Include your OS and hardware info
- Include model details if related to AI features
